waddle project
===================

Read the `Documentation <https://camptocamp.github.io/c2cgeoportal/2.5/>`_

Checkout
--------

.. code::

   git clone git@github.com:camptocamp/waddle.git

   cd waddle

Build
-----

.. code::

  ./build

Run
---

.. code::

   docker-compose up -d

.. Feel free to add project-specific things.
